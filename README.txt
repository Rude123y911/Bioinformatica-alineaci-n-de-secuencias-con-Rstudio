Este programa calcula la óptima alineación dadas dos secuencias
Para poder ejecutar el programa primero se debe tener instalada la librería "stringr"
Y luego se deben correr todas las funciones para que el programa funcione apropiadamente

Al momento de ingresar las secuencias se debe ingresar las de mayor longitud de primero y si son de la misma longitud no importa el orden
